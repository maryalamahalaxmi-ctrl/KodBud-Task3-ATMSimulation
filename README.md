# KodBud Task 3 - ATM Simulation

This project is developed using Java.

## Project Description
The ATM Simulation System is a console-based Java application that simulates basic ATM operations. Users can check account balance, deposit money, withdraw money, and exit the ATM system using a menu-driven program.

## Features
- Check account balance
- Deposit amount
- Withdraw amount
- Validate insufficient balance
- Menu-driven ATM system
- Exit option

## Technologies Used
- Java
- VS Code
